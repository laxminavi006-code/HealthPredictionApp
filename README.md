# 🏥 Health Prediction App

A Flask-based web application that stores patient health data and predicts possible health risks using Google Gemini AI and rule-based logic.

---

## 🚀 Features
- Add patient details (name, DOB, email)
- Store medical data (glucose, haemoglobin, cholesterol)
- AI-powered health risk prediction
- SQLite database storage
- Edit and delete patient records

---

## 🛠️ Tech Stack
- Python
- Flask
- SQLite
- Google Gemini AI
- HTML/CSS (Jinja templates)
